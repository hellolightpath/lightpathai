**The 2am test:** If a user at 2am, in their darkest moment, encounters any element ŌĆö does it feel like a hand on their shoulder?

## COLOR PHILOSOPHY

Every color in the LightPathAI palette carries emotional weight. Nothing is decorative ŌĆö each family serves a purpose in the journey from grief toward light.

| Color Family | Emotional Role |
|---|---|
| **Purple** | The anchor. Trust, stability, the thread that connects both surfaces. Purple says "we're here." |
| **Lavender** | The breath. Soft enough to recede, present enough to hold. Creates space without emptiness. |
| **Sage** | Growth made visible. Used only where progress is real ŌĆö never decorative. Sage is earned. |
| **Sunrise** | The first warmth after a long night. Energy and joy, but only after the user has taken a step. |
| **Glow** | The ambient warmth that fills a room. Softer than Sunrise ŌĆö the light that's already there. |
| **Bloom** | The flower that grows from difficult soil. Celebration, beauty, the Benefits category. |
| **Blush** | The gentlest warmth on skin. Softer than Bloom, used in the sky gradient and tinted backgrounds. |

These colors work together like a sky at dawn ŌĆö each one has its moment, and none overpowers another.

## MOTION & ANIMATION

### Principles
All motion is organic and gentle. Use `ease-in-out` or `ease-out` ŌĆö never linear for UI elements. Background animations run 8ŌĆō22s for subtlety. Interactive transitions: 0.3ŌĆō0.6s.

### Scroll Reveal System
- **Base:** `opacity: 0; transform: translateY(16px); transition: 0.6s ease-out`
- **Variants:** `.reveal-scale` (scale 0.96), `.reveal-left` (translateX -16px), `.reveal-right` (translateX 16px)
- **Revealed state:** `opacity: 1; transform: translateY(0) scale(1)`
- **Stagger:** Children get 80ms incremental delay (0, 80, 160, 240, 320, 400ms)
- Uses IntersectionObserver with `threshold: 0.1`, `rootMargin: 40px 0px -30px 0px`

### Hero Entrance
- `heroEntrance`: 0.65s ease-out, translateY 12px ŌåÆ 0. Children stagger at 0.05s, 0.15s, 0.25s, 0.35s, 0.4s, 0.45s.

### Living Sky (Hero Background)
- 9 drifting orbs with `orbDrift1ŌĆō4` animations (9ŌĆō16s). Complex multi-keyframe translate + scale paths.
- `skyBreathe`: 8s breathing center glow, opacity 0.75ŌĆō1, scale 1ŌĆō1.05.
- `skyShimmer`: 10s gradient sweep. `lightRaysDrift`: 20s diagonal light ray movement.
- `sparkleFloat1ŌĆō2`: 6ŌĆō8s floating sparkle particles with upward drift.

### Impact Section (Dark)
- `blobDrift1ŌĆō3`: 18ŌĆō22s subtle translate and micro-scale on gradient blobs.

### Feature Section
- `featureGlow1ŌĆō2`: 14ŌĆō18s warm glow orb drifts at 0.6ŌĆō0.7 opacity.

### Clouds
- `cloud-drift-right`: translateX -60vw ŌåÆ 110vw. `cloud-drift-left`: reverse direction.

### UI Effects

### Social Media Do's and Don'ts

**Do:** Use the Living Sky palette as the emotional signature. Keep imagery soft and watercolor-like. Use Lora for headlines, Inter for supporting text. Let whitespace breathe. Include the Bloom Mark ŌØ¢ subtly.

**Don't:** Use harsh colors, solid red, or high-contrast neon. Don't use exclamation marks near grief content. Don't crowd the canvas. Never use stock photos of crying people. No dark humor. No urgency language ("Act NOW!", "Don't miss out!").

**Tone on social:** Same as the platform ŌĆö gentle, honest, present tense. "You" and "we." Acknowledge pain without dwelling. Point toward light without minimizing darkness. Every post should pass the 2am test.

### Image Style
Soft watercolor-inspired illustrations in the LightPathAI palette. Figures should be abstract/silhouetted rather than detailed. Landscapes: lavender skies, golden paths, soft horizons. See `lightpath-watercolor.png` as reference. Prefer warm, misty atmospheres over sharp, clinical imagery.
